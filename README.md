# ebay-clone
This project is a clone of eBay, aimed at providing a platform for buying and selling goods online.

## Features
- User authentication
- Product listings
- Shopping cart functionality
- Payment processing

## Tech Stack
- Node.js
- Express
- MongoDB

## Usage
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the application: `npm start`
